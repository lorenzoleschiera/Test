Are you ready for this adventure?

You can "Start" a new game from this main menu.
At the "Start" clicking, you are called to set the number of characters to shoot and then you can start the game clicking on "Play!" button.
As soon as the game page is loaded, you are on the rooftop.
From your overview, you can change the direction pointed by the viewfinder by simply move the mouse. After that you have selected a victim and you have positioned the viewfinder, you can shoot with left click. The bullet is fired.
I hope that you have a good eyesight!
At the moment in which the bullet is fired, the time go slowly so you can follow the bullet's trajectory and see if you hit the target.
If you have been precise, the hitted body part of the target will turn red.

According to the hitted body part, points are awarded to you . You earn 2000 points for head, 1000 for body, 400 for each part of the limbs and 200 for hands or feet.
You win when all victims are died.
For every bullet that you don't fired, you earn 1000 points more.
When your weapon is empty or you kill everybody, the game is ends and you can restart a new game.

If you prefer to set a different game parameters you can click on the gear icon from the main screen of the game and set a daytime, the gravity value or the placement on which you will positioned to shoot yours enemys.
Finally, is also possible to set the default values, to cancel the changes made or to save them.
Always from the main screen of the game you can click on the home icon to return on the main menu.
